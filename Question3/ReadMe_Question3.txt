How To Run
--------------------------------------------------------------------
Use following command to run 

$ hadoop jar Question3.jar Question3 /yelpdata/review.csv /yelpdata/business.csv /kxp153130/Question3/output /kxp153130/Question3/intermediate

input file: /yelpdata/data.csv /yelpdata/business.csv
intermediate directory: /kxp153130/Question3/intermediate
Output directory: /kxp153130/Question3

----------------------------------------------------------------------

Output of the program is there in a "Output" file. You can use Notepad++ to open the
same file to see the output.
